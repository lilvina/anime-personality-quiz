# anime-personality-quiz

A fun, interactive terminal-based anime personality quiz built in Python.
A series of multiple-choice to answer from and discover which anime character matches your personality.

This project uses Unicode emoji code points and colorful terminal output to create an immersive experience.

# Features
- Interactive multiple-choice questions
- Anime character personality matching
- Unicode emoji support
- Colorful terminal UI with ```colorama```

# Requirements
- Python 3.7+
- ```colorama``` package
Install dependencies:
- ```pip install colorama```

# How to Run
1. Clone/download the project
2. Navigate into the project folder:
```cd anime-personality-quiz```
3. Run the quiz:
```python anime_quiz.py``` or ```python3 anime_quiz.py```
